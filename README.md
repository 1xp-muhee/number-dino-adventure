# 숫자 탐험대

5살 아이가 0부터 10까지 숫자를 익히는 모바일 웹입니다.

구성:
- 숫자 카드 학습
- 미션 퀴즈
- 공룡 + 블록 보상 연출
- Web Speech API 기반 한국어 숫자 읽기
- Three.js 기반 가벼운 보상 장면

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## GitHub Pages 배포

이 저장소는 GitHub Pages용 base 경로(`/number-dino-adventure/`)가 설정되어 있습니다.

```bash
npm run deploy
```
